FOTOS DE JUGADORES
==================

Suelta aquí una imagen por jugador con el nombre exacto del id + .jpg.
Cuando el archivo exista, la app la muestra automáticamente (en el avatar y en
la carta tipo FIFA). Si no existe, se mantiene la silueta dorada — sin errores.

Recomendado: foto cuadrada o vertical, cara/torso centrado, ~400x400 px.

  ronaldo.jpg      -> Cristiano Ronaldo (CR7)
  mbappe.jpg       -> Kylian Mbappé
  james.jpg        -> James Rodríguez
  vini.jpg         -> Vinicius Jr.

IMPORTANTE (derechos): usa fotos con licencia libre (por ejemplo Wikimedia
Commons / Creative Commons) o tuyas. Evita fotos con copyright de agencias.
